# Заменитель текста

Настольное приложение для **массовой замены текста** в документах и пакетного
переименования файлов и папок. Поддерживает Word, Excel, PDF и AutoCAD.

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![Platform](https://img.shields.io/badge/OS-Windows-lightgrey)
![UI](https://img.shields.io/badge/UI-Sun%20Valley-2f6fed)

## Возможности

- **Замена текста внутри файлов:**
  - Word `.docx` — основной текст, таблицы, колонтитулы;
  - Excel `.xlsx` — ячейки всех листов;
  - PDF `.pdf` — текст (тройной запас: PyMuPDF → pikepdf → pypdf);
  - AutoCAD `.dxf` и `.dwg` — текстовые объекты TEXT/MTEXT/ATTRIB.
- **Переименование** имён файлов и названий папок.
- **Список задач:** каждая задача имеет собственный текст «найти/заменить»
  и собственную конфигурацию — что именно делать. Конфигурация видна
  прямо в таблице.
- **Всплывающее окно** для добавления и редактирования задачи.
- **Фильтр расширений** для переименования имён файлов (Word, Excel, dwg, pdf).
- **Резервное копирование и откат** всей операции одним нажатием.
- **Современный интерфейс** на теме Sun Valley с цветными иконками.

## Скриншоты

Главное окно — список задач с колонкой конфигурации; всплывающее окно задачи
с разделом «Что делать» и выбором расширений.

## Быстрый запуск (из исходников)

Нужен **Python 3.10+**.

```bash
pip install -r requirements.txt
python main.py
```

Или на Windows просто запустите `start.bat` — он сам поставит зависимости.

## Поддержка `.dwg` (внешние библиотеки)

Работа с `.dwg` требует внешних бинарников **LibreDWG** (и, опционально,
**ACadSharp**). Они **не входят в репозиторий** из-за размера и лицензии.
Как их подготовить — см. раздел 5 инструкции по сборке и `libredwg/README.md`.

Все остальные форматы (Word, Excel, PDF, DXF, имена файлов и папок) работают
без внешних библиотек.
