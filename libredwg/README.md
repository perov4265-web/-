# Папка libredwg

Здесь должны лежать утилиты **LibreDWG** для работы с `.dwg`:

```
dwg2dxf.exe
dxf2dwg.exe
libredwg-0.dll
libpcre2-8-0.dll
libpcre2-16-0.dll
libiconv-2.dll
```

Эти бинарники **не хранятся в репозитории** (большой размер, сторонняя лицензия GPL).
Как их получить — см. `make_exe/ИНСТРУКЦИЯ_СБОРКА_EXE.md`, раздел 5.

Без этих файлов программа работает со всеми форматами, кроме `.dwg`
(Word, Excel, PDF, DXF, имена файлов и папок — работают).
